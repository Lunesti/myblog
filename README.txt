Pour ce projet j'ai utilisé :

- Symfony 5

- Twig

- Bootstrap


Prérequis : 

Voici le lien vers le manager pour modifier les paramètres de connexion à la base de données : https://github.com/Lunesti/myblog/blob/master/.env

Indiquer votre host (le serveur web) (ex : nomserveur.mysql.db),

Votre identifiant,

Votre mot de passe.

N'oubliez pas d'importer la base de données sur le site de votre hébergeur pour récupérer toute les données.
